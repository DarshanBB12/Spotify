# Spotify Clone (HTML & CSS)

This project is a **Spotify Website Clone** created using **HTML and CSS**.
It is a front-end replica of the Spotify interface, built to practice layout design, styling, and modern UI development.

## Project Description

The goal of this project is to recreate the look and feel of the Spotify web player.
It focuses on building a clean and responsive user interface similar to a real music streaming platform.

## Features

* Spotify-like homepage layout
* Sidebar navigation (Home, Search, Library)
* Music playlist section
* Album / song cards design
* Music player UI (play bar design)
* Clean and modern dark theme

## Technologies Used

* HTML5
* CSS3
* Flexbox / Grid

## Project Purpose

This project was created for learning and practice purposes to improve:

* HTML structure
* CSS styling
* Layout and positioning
* UI design skills
* Front-end development fundamentals
